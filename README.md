#  Smart Port Monitoring System

A web-based environmental monitoring dashboard designed to simulate real-time monitoring of port environments.

This project visualizes environmental parameters such as **air quality, water turbidity, and noise levels** to demonstrate how smart monitoring systems can improve sustainable port operations.

---

##  Features

- Real-time environmental dashboard
- Air quality monitoring (PM2.5, CO2)
- Water turbidity monitoring
- Noise level monitoring
- Interactive charts using Chart.js
- Event simulation (pollution scenarios)
- Mini port map visualization
- Animated sensor status indicators

---

##  Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Chart.js
